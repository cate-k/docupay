### Overview

DocuPay is a platform where users can publish their documents and earn ETH when others purchase them.

Currently, users can only upload files to IPFS, and there is no feed where you can find documents posted by others yet. However, I am planning to flesh out this project more in the next few months once I expand my knowledge of Solidity and ReactJS.

It uses IPFS, or the Inter Planetary File System, to upload the files and is deployed on the Rinkeby testnet.
